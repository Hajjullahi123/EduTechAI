# EduTechAI

EduTechAI is a premium IT solutions website built with React and Vite.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm (Node Package Manager)

### Installation
Install the project dependencies:
```bash
npm install
```

### Running the Application
This project is currently a **client-side only** application (React + Vite). There is no separate backend server process required.

To start the development server:
```bash
npm run dev
```

This will launch the application at [http://localhost:5173](http://localhost:5173).

### Building for Production
To create a production build:
```bash
npm run build
```
